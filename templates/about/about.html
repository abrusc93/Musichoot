The Project


This website is a Capstone Project for Computer Science students at Rowan University. We were tasked with creating a social music website, which resulted in MusicHoot. MusicHoot is a social networking application that connects people who share common music interests. Users will use our application to make new friends who listen to similar music and find suggestions for new music that they might be interested in.



Project Goals


  • Use publicly available resources
  • Website that allows users to listen to 30 sec clips of music
  • Allows users to make a free account and login
  • Permit login free browsing before registering
  • The site should provide recommendations for new music to listen to based on inferred preferences
  • Allows users to search for music and create playlists that can be shared
  • Allows users to to see what friends are listening to
  • The website must be made using the Django Framework
  • The Website should be functional on the Chrome Browser


Project Features


  • Web Based: Music Hoot will be available as a web application with both a front-end and a back-end on Django.
  • Login: Will have a login page with fields for a username, password, and email.
  • Listen To Music: Users will be able to listen to 30 seconds clips of songs from Spotify.
  • Recommendations: Users will be able to view recommendations for new music based on previously listened to tracks.
  • Search Music: Users will be able to search for songs to listen to.
  • Music Playlists: Users will be able to create playlists from listening to songs.
  • Friends: Users will be able to make and add friends.
  • Shared Playlists: Users will be able to create sharing permissions for playlists. These permissions include Public which allow anybody to find your playlist. Friends which only allow your friends to find your playlist. Restricted which restricts your playlist viewers to certain people who are invited by the playlist creator.
  • User Profiles: Users will be able to customize their own profiles to display favored songs and playlists. Users will also be able to change privacy settings to show and or hide playlists and liked songs. Users will be able to upvote/downvote tracks and playlists.
